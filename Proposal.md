# <center>I2P2 Final Project Propasal</center>

## <center>第37組 104033217 王哲彥、107060023 黃子恩</center>

1. 遊戲名稱
> _____ / 彈幕射擊遊戲

2. 簡介
    * 玩家操縱一個「自機」，躲避敵方子彈的同時射擊敵機。
3. 操作方式
    * 由方向鍵控制移動
    * 按住shift鍵進入低速移動模式，進行「精密的避彈」
    * 按住Z鍵射擊
    * X鍵發射Bomb
4. 高風險分析
    * 自機移動順暢，要能夠支援↑↓←→Shift 五種按鍵組合，操作起來直觀
    * 敵機要在設定好的時間或順序登場、退場
    * 敵機要能夠發射子彈
    * 碰撞/overlap 判斷機制
    * 自機復活後要有一小段無敵時間
5. 高價值分析
    * 遊戲
        * 背景效果
        * 對話模式(?)
        * 
    * 自機
        * 子彈火力/數量隨著power增加
        * 子彈自動追蹤側方敵機
        * 發射Bomb
        * 擦彈機制
        * 趨前時自動收集掉落物
        * 
    * 敵機
        * 多邊形彈種(如何設計碰撞機制?)
        * Boss

        * 音效配合
